# Lifecell eSIM Number Hunter 🎯

Автоматизований скрипт на Python, який використовує Playwright для пошуку "гарних" (преміальних) номерів на сайті інтернет-магазину lifecell[cite: 2]. Скрипт працює в безпечному режимі зі стандартними затримками, імітуючи реального користувача, та автоматично зупиняється при знаходженні крутої комбінації цифр[cite: 2].

## 🌟 Можливості

Скрипт автоматично витягує згенерований номер, очищає його від зайвих символів і перевіряє за допомогою регулярних виразів на наявність таких комбінацій[cite: 2]:

*   **Повторення:** 3 і більше однакових цифр підряд (наприклад, `777`, `88888`)[cite: 2].
*   **Дублювання коду:** збіг перших 3 цифр номера з кодом оператора (наприклад, `073 073 XX XX`)[cite: 2].
*   **Повторення блоків:** два однакових блоки по 3 цифри після коду (наприклад, `073 X 123 123`)[cite: 2].
*   **Ритм (Пари):** чергування парних цифр у кінці (наприклад, `XX XX 15 15` або `XX 72 72 72`)[cite: 2].
*   **Сходинки:** послідовність з 4 і більше цифр за зростанням чи спаданням (наприклад, `1234` або `9876`)[cite: 2].
*   **Дзеркала (Паліндроми):** дзеркальне закінчення номера (наприклад, `XX XX X 1221`)[cite: 2].

**Додаткові переваги:**
*   **Надійний парсинг:** Скрипт використовує два способи пошуку номера — через основний селектор біля радіокнопки та через блок "У кошику", ігноруючи приховані елементи на сторінці[cite: 2].
*   **Безпечна частота запитів:** Використовується затримка у 2 секунди (`time.sleep(2)`) після очікування події `domcontentloaded`, що дозволяє сторінці повністю завантажити всі елементи кошика без ризику швидкого блокування сервером[cite: 2].

## 🛠 Технології

*   **Python 3.8+**
*   **Playwright** (для автоматизації браузера Chromium та роботи з DOM)[cite: 2].
*   **RegEx** (модуль `re` для парсингу та аналізу номерів)[cite: 2].

## 🚀 Використання

Оскільки сайт lifecell вимагає наявності товару в кошику для доступу до сторінки генерації номера, скрипт працює в напівавтоматичному режимі.

1. Запустіть скрипт у терміналі:
```bash
python hunter.py

```


2. Відкриється вікно браузера Chromium на сторінці каталогу eSIM.


3. У браузері ручками **додайте eSIM у кошик**.


4. Перейдіть до **оформлення замовлення**, щоб на екрані з'явився випадковий номер.


5. Поверніться у термінал і **натисніть `Enter**`.


6. Скрипт перехопить керування і почне автоматично оновлювати сторінку кожні 2-3 секунди, виводячи результати в консоль.


7. Після знаходження "гарного" номера скрипт виведе `🎉 БІНГО!` і зупинить цикл, залишивши вікно браузера відкритим для того, щоб ви могли успішно оформити замовлення.
